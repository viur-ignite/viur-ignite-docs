			<script>
				$(function() {
					// active smooth scroll on all links with href="#..."
					// and log the status in the console
					$('a[href^="#"]').scrollAnchor({
						speed: 900,
						onScrollStart: function() {
							console.info('started scroll');
						},
						onScrollEnd: function() {
							console.info('finished scroll');
						}
					});
				})
			</script>

Fallback!

It's recommend to set always href="#..." as fallback if JavaScript is disabled.
